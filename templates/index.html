Termo de Consentimento para Tratamento de Dados Pessoais

O sistema Combustex poderá coletar e armazenar os seguintes dados pessoais:

  • Nome de usuário
  • Senhas
  • Outros dados pertinentes ao uso do sistema e que sejam necessários para o atendimento e prestação de serviços.

Finalidade do Tratamento de Dados

Os dados coletados serão utilizados para identificação e autenticação do usuário, registro e acompanhamento de transações e serviços, e cumprimento de obrigações legais e regulatórias.

Compartilhamento de Dados

Os dados pessoais poderão ser compartilhados com terceiros apenas quando necessário para a prestação dos serviços do Combustex, com o consentimento do usuário, ou para cumprimento de obrigações legais.

Segurança e Armazenamento dos Dados

O sistema Combustex adota medidas técnicas e administrativas de segurança para proteger seus dados pessoais contra acessos não autorizados.
